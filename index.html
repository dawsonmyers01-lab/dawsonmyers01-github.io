Launch checklist (no errors)

This is the “make it work every time” list.

Repo + Pages settings

  • Repo name: dawsonmyers01.github.io
  • Settings → Pages → Deploy from branch
  • Branch: main • Folder: (root)
  • Homepage file exists: index.html

New post steps

  1. Create: lowercase + hyphens + .html
  2. Add a new <li> at the top of .post-list
  3. href must match filename exactly
  4. Check live site and click the post link

Why 404 happens: the link points to a file name that does not exist, or the case does not match.
