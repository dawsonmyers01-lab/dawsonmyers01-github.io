Week 1: Going live on GitHub Pages

January 25, 2026

This week was about building the foundation and shipping it live: setting up GitHub, using AI as a coding partner, creating the newsletter files, and deploying to GitHub Pages.

What we covered

  • Getting started with GitHub (account + repository)
  • AI as a coding partner (asking for specific files and requirements)
  • Building the newsletter (index, about, posts, and styling)
  • Going live (Pages settings + verifying links)

The most important thing I learned

Publishing is two steps: (1) create the post file, then (2) add it to the homepage post list. If you skip step 2, the post exists but nobody can find it.

What I will improve next

I want to keep posts practical and short, and focus on repeatable workflows: study prompts, planning templates, and “AI as a helper” systems that actually save time.
